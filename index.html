Academic Pages - 学术个人网站模板

Academic Pages 是一个开箱即用的 GitHub Pages 模板，专为学术个人网站设计。

这个网站前端页面由 Academic Pages 模板驱动，托管在 GitHub Pages 上。GitHub Pages 是一项免费服务，网站通过存储在 GitHub 仓库中的代码和数据构建并托管，当仓库有新提交时会自动更新。

此模板派生自 Michael Rose 创建的 Minimal Mistakes Jekyll 主题，然后扩展以支持学术人员需要的内容类型：出版物、演讲、教学材料、作品集、博客文章和动态生成的简历。你现在就可以分叉此模板，修改配置和 Markdown 文件，添加自己的 PDF 和其他内容，免费拥有自己的网站，且没有广告！

数据驱动的个人网站

与许多其他基于 Jekyll 的 GitHub Pages 模板一样，Academic Pages 让你能够将网站内容与其形式分离。你的网站内容和元数据保存在结构化的 Markdown 文件中，而其他各种文件构成了主题，指定如何将这些内容和元数据转换为 HTML 页面。

你在公共 GitHub 仓库中保存这些不同的 Markdown（.md）、YAML（.yml）、HTML 和 CSS 文件。每次向仓库提交并推送更新时，GitHub Pages 服务都会基于这些文件创建静态 HTML 页面，这些页面免费托管在 GitHub 的服务器上。
